# ATS-ETS2-Shift-Inhibitor
Shift inhibitor using an Arduino and the scs-sdk

## Everything here is provided AS IS, No support will be given!

The provided stl files are for the PXN A7 shifter.  The motor is held in place with hot melt glue(easily applied and removed)
The rack needs to be glued into the slide.

Uses a N20 1,000rpm 6v motor, driven at 9v.

The plugin is written using a linux serial library.  Winblows is not my concern!

The include directory from the scs-sdk needs to be copied into the plugin folder to compile it.
